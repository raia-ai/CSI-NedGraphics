Separation Tools
1. Set Sample Number- Set the number of colors for separation
2. Extract Colors
3. Pick Point- Click to pick colors using one precise pixel selection
4. Pick Circles- Click and drag to combine colors inside the circle to make one color sample

6. Next Color- Keep selected to automatically move to the next color in palette
7. View Samples- Click to display the sample for each channel
8. Quality- Click to select the type of separation method suitable for the file
9. Dither Percentage- Option to control the dither percentage while separating a scan
10. Auto Clean- Click on to have file automatically cleaned on Process
11. Auto Smooth- Click on to have file automatically smooth on Process
12. Check List- Preferences for processing
13. Process- The final step in creating your final separation, click on this to process the work
    you have done.

Cleaning Toolbar
1. Clean Under Pencil- Clean area direction under the cursor
2. Pencil Size- Change pencil size
3. Zone Cleaning- Clean larger area using a square selection
4. Lasso Cleaning- Click and drag to clean larger area of your own shape
5. Polyline Cleaning- Click multiple points then double click to connect to clean an area of
   your own shape
6. Global Cleaning- When a cleaning method and color is selected, click on this tool to
   universally clean the image
7. Selection by Color- Click to clean an area connected by one color
8. Filling- Click on a color to fill with selected color
9. Replace Color- Open dialogue box to select colors to be cleaned by replacing them with
   another color
10. Clean Colors- Keep selected to allow you to use a cleaning method to clean the file
11. Cleaning Method- A cleaning method must always be selected to determine what type of
    cleaning needs to be done to the file
12. Tolerance- Determine the number of combined pixels you would like to clean at one time
13. Global Smoothing- Smooth the edges of your entire design
14. Multiple Selection- With palette open, select this icon and choose 2 or more colors to
    clean at the same time
15. Pick Colors- Click to select then click on any color in design to select to clean

Display Toolbar
1. View Rulers- To show rulers
2. Show Repeat Boundaries- View limits of your repeat
3. Tape Measure- To measure the distance between 2 points, click starting point, drag and let go
    at ending point, the distance is recorded on the status bar
4. Show or Hide Palette- Click on to display palette off to hide it
5. View Films- Displays selected color in black while all other colors are white
6. Select Color- Click on a color in your palette to activate it
7. Designate Stencil- Click on icon then click on colors in the palette that you want to protect
8. Stencil- Activates the designated stencil colors, click off to deactivate yet still keep them in
   memory
9. Display Stencil Colors- Shows the protected colors in the design as a checker board so you can
   identify them
10. Real Time Stencil- When clicked on, colors that are stenciled show as you work
11. Drawing Tool Options- Keep this box open when working with drawing tools for more options
      including pen size etc.
12. Show or Hide Layers Dialog- View and manage layers
13. Hand Tool- May move around the design to a chosen area
14. Zoom Area- To zoom in on a specific area, click on icon, then drag box to define the area. Click
     the mouse button inside to finish
15. Magnifier- Magnify the section in your design that is under your cursor
16. Atlas Manager- View and manage list to linked atlases.
17. Color Grouping- Arrange colors into groups
18. Two Color Grouping
19. Color Rules- Determine the outcome of stamping colors on top of each other
20. Tracing- Use this icon to trace a file. New Tracing opens dialog box to browse to image file.
21. Move Tracing- Click inside the document and drag to move the tracing behind the design
22. Select Tracing Color- Click on button and click on tracing to add colors from tracing into your
     color palette
23. Grid- Click to display a grid on your image that represents each individual pixel, you can see
     this by zooming in
24. Cross- Click this button in order to obtain a cross-line for accurate drawing                   7
25. 3D View- This function displays the current design image rendered onto a predefined 3D

Color Tools

1. Exchange Colors- Click on first color chip, and then click the second color chip to change
   the position in the palette
2. Swap Colors- Click on first color chip, and then click the second color chip to change the
   position in the design.
3. Merge Colors- Click on first color chip, and then click the second color chip to merge colors
   together by replacing with second color.
4. Copy Color- Click on the color you would like to copy, and then drag and drop it into the
   color you want to replace; both colors will still be in the palette
5. Move Color- Click on the color you want to move, then click on the color that you want it to
   move to.
6. Merge Equal Color- Click to automatically combine all colors that have the same color code.
7. Unify Palette- View and work with a unified palette for all layers
