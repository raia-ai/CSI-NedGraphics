# Print Studio Pro Toolbar Reference

NedGraphics Print Studio Pro provides professional-grade toolbars for print textile design. This reference documents all available toolbars and their functions.

## Standard Toolbar

**New** - Begin a new print design project.

**Open** - Open existing Print Studio Pro files.

**Save** - Save file with all layers and separations.

**Print** - Print document using current settings.

**Print Preview** - Preview layout before printing.

**About** - Display software version information.

**Help Mode** - Access comprehensive software documentation.

## Zoom Toolbar

**Zoom In** - Increase magnification for detailed editing.

**Zoom Out** - Decrease magnification for overall pattern view.

Allows precise control over viewing scale for both detail work and overall pattern assessment.

## Edit Toolbar

**Cut** - Remove selection and copy to clipboard.

**Copy** - Copy selection to clipboard.

**Paste** - Paste clipboard contents.

**Undo** - Cancel last modification.

**Redo** - Restore last undone action.

**Undo All** - Revert multiple actions.

**Redo All** - Restore multiple actions.

## Module Toolbar

**Load Scan** - Link to Color Reduction and Cleaning Pro for scan import.

**Cleaning** - Link to Color Reduction and Cleaning Pro for color separation.

**Designing** - Link to Design and Repeat Pro for pattern creation.

**Coloring** - Link to Easy Coloring Pro for colorway generation.

**Layout** - Link to Storyboard and Cataloguing Pro for presentations.

## Print Design Tools

**Repeat Tools** - Create seamless repeats with straight, half-drop, and custom options.

**Color Separation** - Separate designs into individual color layers for screen printing.

**Cleaning Tools** - Remove stray pixels and clean up artwork.

**Colorway Tools** - Generate multiple color variations efficiently.

**Template Tools** - Create and apply presentation templates.

Print Studio Pro delivers professional print design capabilities with advanced color management, seamless repeat creation, and production-ready output. The integration with other NedGraphics modules enables a complete workflow from concept through final production files.
