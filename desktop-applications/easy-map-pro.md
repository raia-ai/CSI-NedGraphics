Standard Toolbar
1.  New- Click to begin a new project
2.  Open- Open an already existing file
3.  Save - Saves file as current type
4.  Print- Prints your document using current settings
5.  Cut- Click to remove the image of the active window and copy it to the clipboard
6.  Copy- Click to keep the image of the active window and copy it to the clipboard
7.  Paste- Click to paste the contents of the clipboard
8.  Save Mapped Photo- Export the current project as an image, which will not be
    able to be edited in Easy Map.
9.  Margins and Background- Create a frame to surround the image
10. Crop- Eliminate an outer area of the image
11. Scroll- Click and drag to move the picture in any direction

    Segment Toolbar
    1.New Layer- Create a new layer where zones of mapping might overlap each other
    2.Delete Layer- Remove a layer from the project
    3.Layer Properties- View and edit the information of any layer
    4.Segment Selection- Switch to segments selection mode to select segments of the project
    5.Transform- Allows you to enter an edit mode for the current segment
    6.New Segment- Create a new segment, a surface to be mapped, by drawing a line chain
      around the area
    1.Free Hand- Create a segment using the paintbrush method
    2.Free Hand Delete- Redefine/ modify a segment that has been defined with the freehand
      drawing too
    1.Convert to Color- Can convert a new segment to paintbrush mode, so that it can be
      modified more accurately with the freehand tools.
    1.Delete Segment- Clear the current segment
    2.Segment Properties- Display overview of all segments
    3.Polyline Selection- Activate selection mode
    4.Add Selection- Add the polyline selection that has been defined to the currently selected
      segment
    1.Extract Selection- Remove the polyline selection from the currently selected segment
    2.Extract Selection from all Segments- Remove the polyline section from all segments
    3.Freehand Bezier- Define a segment by means of a freehand bezier curve
    4.Standard Bezier- Define a segment by means of a standard bezier curve
    5.Middle Point Bezier- Define a segment by means of a middle point bezier curve
    6.Magnetic Lasso- Draw a freehand lasso that snaps to the lines of an image

Grid Tools
1.   Grid Selection- Allows you to select a particular grid when in grid mode
2.   Hermite Grid- Associate a hermite grid with current segment; these are flexible and can be
     manipulated in multiple directions
3.   Floor Grid- Create a floor grid for current segment. Its control points cannot be turned into
     bending or connecting points
4.   Slave Grid- Reusing an existing grid that has been defined for another segment, so that
     many of its settings have already been determined
5.   Freehand Grid- Creates a grid and all of its border lines and intersection points from scratch
6.   Flip Grid- Reflect current grid on a vertical axis
7.   Mirror Grid- Reflect current grid on a horizontal axis
8.   Grid Properties- Display global overview of all grids present in project
9.   Patch Properties- Opens the patch properties dialog to set the density for each patch
10. Reference Point- Modify the coordinates of the reference point, or origin, or the fabric
11. View All- Display all grids; use the arrow to the right of the icon to view all grids on current
     layer or on all layers
12. View Segments- Allows you to view all segments while still working in grid mode                  3

Design Tools
1.   Assign Fabrics- Select fabrics to be mapped onto segments/groups
2.   Assign Colors- Select colors to be mapped onto segments/groups from color
     channels or atlas
3.   Mapping Properties- customize mapping parameters for each segment/group
4.   Resize Design- Adjust scale of the fabric that has been mapped
5.   Rotation Angle- Rotate fabric to a chose angle and click set to update the project
6.   Mirror- Reflect the fabric on a vertical axis
7.   Flip- Reflect the fabric on a horizontal axis
8.   Set Placing Point- Set or modify the mapping origin

Undo/ Redo
1.  Undo All - Click to undo a series of actions
2.  Undo - Click to cancel the last modification made to the design during the current
    session
3.  Redo - Click to cancel the effect of the last undo command
4.  Redo All - Click to redo all actions altered in Undo

Easy Map Managers
1.  Layers Manager- Opens the layer properties dialog to modify properties of the
    various layers
2.  Groups Manager- Sort segments into groups
3.  Designs Manager- Used for fabric and color assignment
4.  Shadows Manager- Opens a shadow properties dialog to set the shadow for
    each segment/group
5.  Mapping Manager- Customize mapping parameters
