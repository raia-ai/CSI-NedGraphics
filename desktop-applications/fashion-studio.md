# Fashion Studio Toolbar Reference

NedGraphics Fashion Studio provides specialized toolbars for fashion and apparel design. This reference documents all available toolbars and their functions.

## Standard Toolbar

**New** - Begin a new fashion design project.

**Open** - Open existing Fashion Studio files.

**Save** - Save file with all design data and layers.

**Copy** - Copy current selection to clipboard.

**Print** - Print document using current settings.

**About** - Display software version and licensing information.

**Help Mode** - Access full software documentation and guides.

**Hide/Show Objects** - Toggle toolbar visibility to maximize workspace.

## Zoom Toolbar

**Zoom In** - Increase magnification for detailed work.

**Zoom Out** - Decrease magnification for overall view.

Allows you to zoom in and out and display design at various scales for precision work.

## Edit Toolbar

**Cut Image** - Remove design and copy to clipboard.

**Copy Image** - Keep design and copy to clipboard.

**Paste Over Image** - Paste clipboard contents over existing design.

**Undo** - Cancel last modification.

**Redo** - Restore last undone action.

**Undo All** - Revert multiple actions.

**Redo All** - Restore multiple actions.

## Module Toolbar

**Cleaning** - Link to Color Reduction and Cleaning Pro.

**Designing** - Link to Design and Repeat Pro.

**Coloring** - Link to Easy Coloring Pro.

**Layout** - Link to Storyboard and Cataloguing Pro.

## Fashion Design Tools

**Drawing Tools** - Create technical flat sketches and garment illustrations.

**Symmetry Tools** - Draw symmetrical garments automatically.

**Stitch Lines** - Add topstitching and construction details.

**Fill Tools** - Fill garment areas with colors, fabrics, or patterns.

**Garment Library** - Access pre-built garment templates and components.

**Tech Pack Tools** - Create technical specification sheets.

Fashion Studio enables designers to create professional technical flats, apply fabric swatches, generate colorways, and prepare complete tech packs for production. The integration with other NedGraphics modules ensures a seamless workflow from concept through production documentation.
