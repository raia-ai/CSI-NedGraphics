Main Tools
1.   Select Tool
2.   Insert Objects Toolbar- Displays options to insert a picture, 3D, fabric, article, material or
     CDS file object
3.   Straight Line- Draw a straight line
4.   Insert Text- Click and drag to create a text box
5.   Apply Fabric- Select Apply Fabric button and then click and drag a fabric onto an article or
     3D mapping to apply it
6.   Mapping Origin- Change where the origin of a fabric is positioned on an article or mapping.
7.   Open Browsers Manager- Displays list of open books and allows you to add multiple books
     of the same type, from different file locations
8.   Object Properties- Displays object properties toolbar at bottom of page
9.   Object Manager- Displays object manager to left of window, which lists all elements of each
     page
10. Atlas Manager- View and manage which atlases are open
11. Rulers- Click to show rulers along side the board
12. Grid- Display a grid to make layout-related operations easier

Arrange Toolbar
1.   Group- Select two objects or more and group them
2.   Ungroup- Ungroup a set of selected objects that were previously grouped
3.   Ungroup all Page- Ungroup all groups of objects on page
4.   Change Position- Displays Z-Order options to move objects in front or behind
     each other
5.   Alignment Toolbar- Displays Alignment options to line objects up horizontally or
     vertically
6.   Distribution Toolbar- Displays Distribution options to arrange three or more
     objects evenly along the x or y axis of the page.
7.   Arrange Objects in Page- When selected, objects can be aligned to the
     parameters of the page rather than to each other.
8.   Window Arrangement Toolbar- Arrange storyboard windows

Distribution Toolbar
When three objects are selected, the distribution tools arrange them evenly on the page
1.    Distribute from Top- Distributes three objects evenly from top of page with respect to top edges
2.    Distribute Horizontally from Center- Distributes three objects evenly from the centre of page
      with respect to center of objects
3.    Distribute from Bottom- Distributes three objects evenly from bottom of page with respect to
      bottom edges
4.    Distribute from Left- Distributes three objects evenly from left of page with respect to left edges
5.    Distribute Vertically from Center- Distributes three objects evenly from the centre of page with
      respect to center of objects
6.    Distribute from Right- Distributes three objects evenly from right of page with respect to right
      edges

Pages Manager
1.  New Page- Add a page to the end of the presentation
2.  Insert Page- Add a page to a chosen location within the presentation
3.  Duplicate Page- Create a duplicate page and add to presentation
4.  Delete Page- Delete the page from the presentation
5.  Page Properties- Opens the object properties dialog
6.  Formatting Page- Takes you to the formatting page of the presentation, where
    any changes made will be applied to all pages of the presentation
7.  Go to Page- Allows you to navigate to any page of the presentation
8.  Move Page- Opens the Move page dialog where you can rearrange pages
