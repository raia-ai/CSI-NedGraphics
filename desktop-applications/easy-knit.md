# Easy Knit Toolbar Reference

NedGraphics Easy Knit provides comprehensive toolbars for creating knitted textile designs. This reference documents all available toolbars and their functions.

## Standard Toolbar

**New** - Click to begin a new knit design project.

**Stripes** - Build stripes and plaids for knitted patterns.

**Ombre Generator** - Create gradient color effects in knit designs.

**Open** - Open an already existing Easy Knit file.

**Save** - Saves file in current type with all design data.

**Print** - Prints your document using current print settings.

**Print Preview** - Click to open print menu and preview layout before printing.

**About** - Displays the latest software data including version numbers.

**Help Mode** - Provides full detailed guide about the software.

## Zoom Toolbar

**Zoom In** - Increases magnification to view design details.

**Zoom Out** - Decreases magnification to view more of the design.

The Zoom Toolbar allows you to zoom in and out and display more or fewer repeats of your knit pattern.

## Edit Toolbar

**Cut Image** - Click to remove the design of the active window and copy it to the clipboard.

**Copy Image** - Click to keep the design of the active window and copy it to the clipboard.

**Paste Over Image** - Click to paste the contents of the clipboard over the existing design.

**Undo** - Click to cancel the last modification made to the design during the current session.

**Redo** - Click to cancel the effect of the last undo command.

**Undo All** - Click to undo a series of actions.

**Reinstate Last But One** - Click undo all the way back to the change you wish to make, then click reinstate and click redo all.

**Redo All** - Click to redo all actions altered in Undo.

## Module Toolbar

The Module Toolbar shows the modules currently accessible for linking with Easy Knit.

**Load Scan** - Click to link to Color Reduction and Cleaning Pro for importing scanned artwork.

**Cleaning** - Click to link to Color Reduction and Cleaning Pro for color separation and cleanup.

**Designing** - Click to link to Design and Repeat Pro for pattern creation.

**Coloring** - Click to link to Easy Coloring Pro for colorway generation.

**Layout** - Click to link to Storyboard and Cataloguing Pro for presentation preparation.

## Functions Toolbar

**Stitch Repeat** - Set the repeat size for the knit pattern.

**Stitch Repetition** - Configure repeat intervals in the design.

**Stitch Symmetry** - Set symmetry options for balanced patterns.

**Automatic Copy** - Automatically copy design elements across the pattern.

## Knit Design Tools

**Draw Stitches** - Create knit stitches in the design area.

**Stitch Selection** - Select individual stitches or stitch groups for editing.

**Color Control** - Provides color statistics for the entire design and each individual line of the project.

**Tracing** - Use this to trace a file. Opens dialog box to browse to source images for recreation.

**Select Whole Design as Motif** - Automatically selects entire design as a motif and places it in the motif library.

**Rotation** - Opens a dialog box that allows you to rotate the design.

**3D View** - View the current knitting project mapped onto predefined 3D objects to visualize the finished product.

## Workflow Integration

Easy Knit integrates seamlessly with other NedGraphics modules through the Module Toolbar, enabling a complete workflow from scan import through color reduction, design creation, coloring, and final presentation layout. This integration eliminates the need to export and import files between applications, maintaining design fidelity and reducing production time.
